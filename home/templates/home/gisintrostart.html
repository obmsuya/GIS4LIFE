Learning Objectives

This course will provide foundational knowledge of a GIS and show how it can be used to acquire geographic knowledge. You will explore GIS maps and use tools to gather information that you can use for decision-making at work or in your daily life.

After completing this course, you will be able to perform the following tasks: Define GIS, Differentiate between vector and raster data, Navigate a GIS map, Use tools to access feature information and Create a location query and an attribute query.
