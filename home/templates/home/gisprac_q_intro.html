INTRODUCTION

About this section

This online course provides practical introduction to QGIS; a free, open-source Geographic Information System. It dive deep and take a comprehensive look at QGIS and the basic GIS concepts. You will learn through lecture videos, exercices, quizzes and social interaction.

QGIS is a popular open-source GIS with advanced capabilities. The practical has been tailored to QGIS 2.12, this may be different to the version you may be using and therefore may look different, but the concepts and tasks will be similar. You are adviced to install and use similar version to avoid confusion. Click the dowload button bellow for the setup

After completing this section, you will be able to correctly identify the main elements of the screen in QGIS and know what each of them does, and load a shapefile into QGIS. You will learn the principles underpinning vector-based geographic data analysis and learn to undertake tasks such as sourcing, loading and interrogating data; creating professional map outputs and preparing geographic datasets.

Installation


How to obtain QGIS?

QGIS is freely available on Windows, Linux, MacOS X, BSD, and Android. We recommend installing the packages of the released software. For new users we recommend the standalone installers.

Click Here to qgis setup for this Course.

For more advanced QGIS users you can use OSGeo4W packages, which make it possible to install several versions in parallel. The OSGeo4W repository contains a lot of software from OSGeo projects. QGIS and all dependencies are included, along with Python, GRASS, GDAL, etc. (DO NOT INSTALL THIS)

Watch the video below for more undestanding

What is QGIS software?

QGIS is an open source geographic information system software. It runs on Windows, Mac OS X, and Linux. QGIS is a user-friendly GIS, providing common functions and features; also a free, collaborative, cross-platform software that is constantly developing by QGIS community.

First thing first (Installing QGIS)

You can download setup file for QGIS using following link: https://www.qgis.org/en/site/index.html

After downloading the setup file, execute the file and install the software simply clicking "Next" for all dialog options.
